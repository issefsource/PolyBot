# PolyBot 🤖

PolyBot is a trading bot designed to interact with Polymarket data and help automate trading decisions.

## 🚀 Features

* Reads market data
* Analyzes opportunities
* Executes trades automatically (in progress)

## 📦 Setup

1. Install Python (3.10+ recommended)

2. Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Run the bot

```
python bot.py
```

## ⚠️ Notes

* This project is still in development
* Do not use real money without testing
